Interrupt Task 

Implement a Ping-Pong game using LCD and external interrupt.
Algorithm:

1- Display a ball moving on LCD from one end to the other.
2- Two players are playing (one on each side) with two buttons ( the button represents the Ping pong racket )
3- Use external interrupt to indicate whether the button is pressed or not.
4- If the player press the button, the ball should bounce to the other side
5- If not pressed the opposite player wins the game
